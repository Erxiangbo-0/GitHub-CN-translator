# 如何使用内置翻译
  - 下载项目文件的压缩包到本地
  - 解压缩后打开edge浏览器，开启**开发人员模式**
  - 点击**加载解压缩的扩展**，选择你解压缩项目的目录
  - 访问 github.com 即可看到翻译效果，点击扩展图标可切换开关
  - **在内置模式下，只能翻译部分内容，比如导航栏中的模块等**
# 如何接入大模型API
  - 在扩展中点开翻译插件面板
  - 点击AI智能翻译下的**API配置**跳转至配置界面
  - 依次填写模型，API key，API 端点地址等，支持自主修改prompt
  - 点击连接测试，得到成功提示后点击保存
  - 回到GitHub界面刷新即可
  - **由于模型返回需要时间，在界面翻译没有生效时可以刷新几次或者切换界面**
